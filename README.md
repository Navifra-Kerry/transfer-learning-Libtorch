# Requirements
 * OpenCV 4.1.1
 * Libtorch 1.4
 * CUDA 10.2
 * Visual studio 2019
 * Windows 10 

Install with Scripts

```
transfer-learning
 - Scripts
    -install.bat
    
Run install.bat
It will download and install automatically.
```
